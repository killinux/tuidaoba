== Description ==

WP-FlashTime Widget allow you to add a flash clock widget to your wordpress.This plugin include 25 different flash clocks. Check flash clocks number here : http://horoscop2009.org/wp-flashtime
RO : Acest plugin va da posibilitatea de a adauga un widget cu ceas pe blogul wordpress.Pluginul include 25 de tipuri de ceasuri.
Plugin dezvoltat de [horoscop](http://horoscop2009.org) [horoscop  2010](http://horoscop2010.net) si [horoscop](http://horoscopnet.ro) [play sonic games](http://www.techetheatre.org) [free online movies](http://www.vidics.com)

== Installation ==

1. Download the zip file and extract the contents.
2. Make sure you're running WordPress version 2.3 or better. It won't work with older versions.
3. Upload the 'wp-flashtime-widget' folder to your plugins directory (plugins/wp-flashtime-widget).
4. Activate the plugin through the 'plugins' page in WP.
5. See widget options to adjust things like display size, clock style etc...

== Frequently Asked Questions ==

= My theme/site appears not to like this plugin. It's not displaying correctly. =

* In 99% of all cases where this happens the issue is caused by markup errors in the page where the plugin is used. Please validate your blog using [validator.w3.org](http://validator.w3.org) and fix any errors you may encounter.
* The plugin requires Flash Player 9 or better and javascript. Please make sure you have both.
